Выбор необходимых для работы модулей Pure

Сборка

Пакеты сборок кода состоят из всех модулей Pure. Есть два вида сборок фреймворка Pure - для адаптивного дизайна страницы и неадаптивного. Выберите тот пакет, который подходит для вашего проекта.

Имя URL Размер (сжатый)
Пакет для адаптивного дизайна http://yui.yahooapis.com/pure/0.4.2/pure-min.css 4.5KB
Пакет для неадаптивного дизайна http://yui.yahooapis.com/pure/0.4.2/pure-nr-min.css 4.2KB

Альтернативные сети дистрибуции контента (CDN)

Основным хостером для Pure является Yahoo! CDN, где файлы фреймворка Pure доступны по протоколу HTTP и отдельные модули могут быть объединены вместе в одном запросе. В следующей таблице перечислены альтернативные CDN, где Pure также размещён.

CDN URL HTTPS Комбинирование модулей в одном запросе
cdnjs //cdnjs.cloudflare.com/ajax/libs/pure/0.4.2/pure-min.css Да Нет
OSS MaxCDN //oss.maxcdn.com/libs/pure/0.4.2/pure-min.css Да Нет
Staticfile http://cdn.staticfile.org/pure/0.4.2/pure-min.css Нет Нет

Примечание: Если последней версии Pure нет ещё ни на одном из альтернативных CDN, пожалуйста, свяжитесь с разработчиками, чтобы они обновили файлы до последней версии Pure: 0.4.2.

Отдельные модули

Вы также можете подключать различные модули Pure по отдельности.Yahoo! CDN поддерживает обработку комбинированных запросов, так что вы можете посредством комбинированного запроса подключить несколько необходимых вам модулей Pure в одном CSS-файле. Вот пример комбинированного URL, который запрашивает CSS-код основы, сеток и форм: 

<link rel="stylesheet" href="http://yui.yahooapis.com/combo?pure/0.4.2/base-min.css&pure/0.4.2/grids-min.css&pure/0.4.2/forms-min.css">
Имя URL
Основа http://yui.yahooapis.com/pure/0.4.2/base-min.css
Кнопки http://yui.yahooapis.com/pure/0.4.2/buttons-min.css
Формы (адаптивные) http://yui.yahooapis.com/pure/0.4.2/forms-min.css
Формы (неадаптивные) http://yui.yahooapis.com/pure/0.4.2/forms-nr-min.css
Сетки (адаптивные) http://yui.yahooapis.com/pure/0.4.2/grids-min.css
Сетки (неадаптивные) http://yui.yahooapis.com/pure/0.4.2/grids-nr-min.css
Меню (адаптивное) http://yui.yahooapis.com/pure/0.4.2/menus-min.css
Меню (неадаптивное) http://yui.yahooapis.com/pure/0.4.2/menus-nr-min.css
Таблицы http://yui.yahooapis.com/pure/0.4.2/tables-min.css

Создание тем

Pure сделан в минималистском стиле, что позволяет настроить его с вашим собственным CSS-кодом. YUI Skin Builder является отличным инструментом, если вам нужна помощь в разработке собственной темы к CSS-фреймворку Pure. С помощью одного клика, YUI Skin Builder способна генерировать все CSS-стили, необходимые для изменения внешнего вида Pure.
